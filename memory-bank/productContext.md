# Product Context

## Why This Project Exists

Saudi real estate brokers need a unified platform that:
1. Lists properties (Aqar-style) with REGA compliance
2. Matches property seekers with brokers (DealApp-style)
3. Manages contacts, pipelines, and transactions
4. Integrates with Saudi government systems (Nafath, Ejar, ZATCA)

## Problems Solved

- Fragmented tools for listing, matching, and CRM
- Manual REGA license validation
- No integrated request–offer flow
- Lack of Arabic-first, RTL-native UX

## User Experience Goals

- Arabic RTL as primary; logical CSS (ps-, pe-, ms-, me-)
- Property cards with SAR pricing, م² area, neighborhood
- Map view with property pins (Google Maps)
- Broker request inbox with neighborhood matching
- Kanban pipelines for leads and transactions
